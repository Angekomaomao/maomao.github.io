# 设置留言板开机自动启动

## 方法1：添加到启动文件夹

1. **按 `Win + R`** 打开运行对话框
2. **输入**：`shell:startup`，按回车
3. **右键点击 `start-ngrok.bat`** → 发送到 → 桌面快捷方式
4. **将桌面快捷方式移动**到刚才打开的"启动"文件夹
5. **重启电脑测试**

## 方法2：使用任务计划程序（更可靠）

1. **按 `Win + R`**，输入 `taskschd.msc`，按回车
2. **点击右侧** → **创建基本任务**
3. **名称**：留言板自动启动
4. **触发器**：选择"当计算机启动时"
5. **操作**：选择"启动程序"
6. **程序**：浏览选择 `c:\Users\HW\Desktop\留言web\start-ngrok.bat`
7. **完成**

## 注意事项

- 开机后会自动弹出命令行窗口
- Ngrok 地址可能每次都不同
- 可以在 http://127.0.0.1:4040 查看当前地址
